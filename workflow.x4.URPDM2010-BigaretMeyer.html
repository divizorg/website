Ranking: ACUTA vs. weighted sum

Short description

This workflow compares the ranking obtained by the ACUTA algorithm with the one obtained with a weighted sum, on the classical “Thierry’s sports car choice” example [BouyssouEtAl2000], created for the URPDM2010 conference in Coimbra, Portugal.

Detailed description

This workflow contains 3 parts:

  • the upper area represents the use of the disaggregation technique ACUTA [BousEtAl2010] to compute partial value functions on basis of a ranking given as an input;
  • the lower area represents a classical weighted sum;
  • the right part compares the output of the two techniques.

This example is discussed in further details in [BigaretMeyer2010].
