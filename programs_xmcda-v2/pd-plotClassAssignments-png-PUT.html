plotClassAssignments-png

Provider:PUT
Version:1.0.0

Description

Visualizes class assignments for given alternatives as an image.

  • Contact: Maciej Uniejewski <maciej.uniejewski@gmail.com>
  • Web page: https://github.com/maciej7777/PrometheeDiviz

Inputs

(For outputs, see below)

  • alternatives
  • categories
  • assignments
  • categories_values
  • method_parameters

alternatives

Description:

Alternatives to consider.

XMCDA related:

  • Tag: alternatives

categories

Description:

Definitions of categories.

XMCDA related:

  • Tag: categories

assignments

Description:

Assignments of alternatives to given classes.

XMCDA related:

  • Tag: alternativesAffectations

categories marks

Description:

Marks of categories (higher mark means better category). Each category need to have unique mark from 1 to C, where C is a number of categories.

XMCDA related:

  • Tag: categoriesValues

method parameters

Description:

A set of parameters provided to tune up the module’s operation.

GUI information:

  • Parameter values can be defined via the GUI or the XMCDA file, by default via GUI.

  • Name: Visualization type

    Information of the chosen type of visualization.

    • Type: drop-down list
    • Possible values:
      • normal (XMCDA label : normal) (default)
      • dashed (XMCDA label : dashed)

XMCDA related:

  • Tag: methodParameters

  • Code:

    <methodParameters>
    <parameter id="visualizationType">
        <value>
            <label>%1</label>
        </value>
    </parameter>
</methodParameters>
